# 🎧 Indian Language Audio Classification

## 📌 Overview

This project focuses on classifying Indian languages from audio signals using Machine Learning / Deep Learning techniques. It processes audio data and predicts the corresponding language.

---

## 🚀 Features

* 🎙️ Audio preprocessing (noise removal, normalization)
* 📊 Feature extraction (MFCC / Spectrogram)
* 🤖 Model training and evaluation
* 🌐 Multi-language classification

---

## ▶️ How to Run

1. Clone the repository:

```
git clone https://github.com/Charankumm/Indian-Language-Audio-Classification.git
```

2. Navigate to the folder:

```
cd Indian-Language-Audio-Classification
```

3. Install dependencies:

```
pip install -r requirements.txt
```

4. Run the notebook or script:

```
jupyter notebook
```
